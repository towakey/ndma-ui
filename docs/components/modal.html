Modals

モーダルは、ユーザーの注意を特定のコンテンツや操作に集中させるためのダイアログです。確認ダイアログ、詳細情報の表示、フォーム入力などに使用されます。

基本的なモーダル

最もシンプルなモーダルの構造です。

基本モーダル 
<!-- トリガーボタン -->
<button class="ndma-btn ndma-btn--primary" onclick="openModal('basicModal')">基本モーダルを開く</button>

<!-- モーダル -->
<div id="basicModal" class="ndma-modal">
  <div class="ndma-modal__overlay" onclick="closeModal('basicModal')"></div>
  <div class="ndma-modal__content">
    <div class="ndma-modal__header">
      <h3 class="ndma-modal__title">基本モーダル</h3>
      <button class="ndma-modal__close" onclick="closeModal('basicModal')">×</button>
    </div>
    <div class="ndma-modal__body">
      <p>これは基本的なモーダルの例です。</p>
    </div>
    <div class="ndma-modal__footer">
      <button class="ndma-btn ndma-btn--secondary" onclick="closeModal('basicModal')">閉じる</button>
      <button class="ndma-btn ndma-btn--primary">保存</button>
    </div>
  </div>
</div>

<script>
function openModal(modalId) {
  document.getElementById(modalId).classList.add('ndma-modal--open');
  document.body.style.overflow = 'hidden';
}

function closeModal(modalId) {
  document.getElementById(modalId).classList.remove('ndma-modal--open');
  document.body.style.overflow = '';
}
</script>

使用ガイドライン

適切な使用場面

  • 確認ダイアログ：削除、保存など不可逆な操作
  • フォーム入力：詳細情報の追加や編集
  • 詳細表示：リスト項目の詳細情報
  • 重要通知：システム警告、メンテナンス情報

アクセシビリティ

モーダルを開いたときは、フォーカスをモーダル内に移動し、閉じたときは元の要素に戻してください。ESCキーで閉じられるように実装します。

<div id="modal" class="ndma-modal" role="dialog" aria-modal="true" aria-labelledby="modalTitle">
  <div class="ndma-modal__content">
    <h2 id="modalTitle" class="ndma-modal__title">タイトル</h2>
    <!-- 内容 -->
  </div>
</div>

<script>
document.addEventListener('keydown', function(e) {
  if (e.key === 'Escape') {
    closeModal('modal');
  }
});
</script>

パフォーマンス

モーダル内に重いコンテンツを配置する場合は、遅延読み込みを検討してください。また、開く前に必要なデータを準備しておくことで、ユーザー体験が向上します。

モバイル対応

モバイルデバイスでは、モーダルが画面全体に表示されるように調整してください。スワイプで閉じられる機能も検討に値します。
