Tables

データを整理して表示するためのテーブルコンポーネント

テーブルはデータを行と列で整理して表示するためのコンポーネントです。災害情報、報告書、統計データなど、構造化された情報の表示に適しています。

<script>
function sortTable(columnIndex) {
  const table = document.querySelector('table');
  const tbody = table.querySelector('tbody');
  const rows = Array.from(tbody.querySelectorAll('tr'));
  
  rows.sort((a, b) => {
    const aText = a.cells[columnIndex].textContent;
    const bText = b.cells[columnIndex].textContent;
    
    if (!isNaN(aText) && !isNaN(bText)) {
      return parseFloat(aText) - parseFloat(bText);
    }
    return aText.localeCompare(bText);
  });
  
  rows.forEach(row => tbody.appendChild(row));
}
</script>

使用ガイドライン

適切なテーブルの選択

  • 基本テーブル：一般的なデータ表示
  • ストライプ：多くの行を読みやすくする
  • ボーダー：セルの区切りを明確にする
  • ホバー：インタラクティブな行操作
  • コンパクト：限られたスペースでの表示
  • レスポンシブ：モバイル対応が必要な場合

アクセシビリティ

テーブルには適切な見出し（<th>）とscope属性を使用してください。

<table>
  <thead>
    <tr>
      <th scope="col">名前</th>
      <th scope="col">年齢</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">田中</th>
      <td>35</td>
    </tr>
  </tbody>
</table>

パフォーマンス

大量のデータを表示する場合は、ページネーションや仮想スクロールを検討してください。
