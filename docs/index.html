クイックスタート

ndma-uiをすぐに使い始める方法：

CDNで利用

<!-- ndma-ui CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/ndma-ui@1.0.0/dist/ndma.min.css">

<!-- テーマクラスをHTML要素に追加 -->
<html class="ndma-theme-default">

NPMでインストール

npm install ndma-ui

基本的なHTMLテンプレート

<!DOCTYPE html>
<html lang="ja" class="ndma-theme-default">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>NDMA UI サンプル</title>
  <link rel="stylesheet" href="path/to/ndma.css">
</head>
<body>
  <div class="ndma-container">
    <header class="ndma-header">
      <h1>災害管理システム</h1>
    </header>
    
    <main class="ndma-main">
      <section class="ndma-section">
        <h2>状況概要</h2>
        <div class="ndma-card">
          <p>現在、特別警報は発令されていません。</p>
        </div>
      </section>
    </main>
  </div>
</body>
</html>

世界観設定

NDMA（National Disaster Management Agency）
日本の国立災害管理機関をモチーフとした、行政システム向けのUIデザイン。
SOPHEX（Special Operations Headquarters Executive System）
特殊作戦司令部執行システム。Night Opsテーマで利用されるHUD風インターフェース。
Night Ops Mode
夜間作戦・災害対応シーンに最適化された、視認性と緊張感を両立した特別テーマ。
