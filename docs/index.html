特徴

🎯 軽量・高速

依存関係のない純粋なCSSフレームワーク。わずか45KBで完全に機能します。

🏛️ 行政UI最適化

日本の行政システム・公的機関向けに最適化されたデザイン言語。

🌙 多彩なテーマ

Default（行政）とNightvision（夜間視認・NERV風）の2つのテーマを標準装備。

♿ アクセシビリティ

WCAG 2.1 AAレベル準拠。すべてのユーザーが利用可能です。

📱 レスポンシブ

モバイルファースト設計。あらゆるデバイスで最適表示。

🎨 カスタマイズ可能

CSS変数ベースで簡単にカスタマイズ。ブランドに合わせて変更。

クイックスタート

ndma-uiをすぐに使い始める方法：

CDNで利用

<!-- ndma-ui CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/ndma-ui@1.0.0/dist/ndma.min.css">

<!-- テーマ属性をHTML要素に追加 -->
<html data-theme="default">

NPMでインストール

npm install ndma-ui

基本的なHTMLテンプレート

<!DOCTYPE html>
<html lang="ja" data-theme="default">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>NDMA UI サンプル</title>
  <link rel="stylesheet" href="path/to/ndma.css">
</head>
<body>
  <div class="ndma-container">
    <header class="ndma-header">
      <h1>災害管理システム</h1>
    </header>
    
    <main class="ndma-main">
      <section class="ndma-section">
        <h2>状況概要</h2>
        <div class="ndma-card">
          <p>現在、特別警報は発令されていません。</p>
        </div>
      </section>
    </main>
  </div>
</body>
</html>

世界観設定

NDMA（National Disaster Management Agency）
日本の国家災害管理機構をモチーフとした、行政システム向けのUIデザイン。
CENTRIS（Central Emergency Response & Intelligence System）
特殊作戦司令部執行システム。Examplesページで操作端末UIモックとして紹介。
Nightvision Mode
夜間視認に最適化された、黒背景 + 蛍光オレンジ + 角丸なしの硬質デザイン。
