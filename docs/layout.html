Layout

グリッド、フレックス、スペーシングのユーティリティ

グリッドシステム

.ndma-grid と .ndma-grid-cols-* を組み合わせて列数を割り当てます。

cols-3 / item 1
cols-3 / item 2
cols-3 / item 3
<div class="ndma-grid ndma-grid-cols-3">
  <div class="ndma-card">...</div>
  <div class="ndma-card">...</div>
  <div class="ndma-card">...</div>
</div>

フレックスユーティリティ

横並び/縦並び、中央寄せ、両端揃えなどをユーティリティで組み立てます。

左 
<div class="ndma-flex ndma-justify-between ndma-items-center">
  ...
</div>

スペーシングユーティリティ

padding / margin を .ndma-p-*, .ndma-m-* で指定できます。

外側: .ndma-card ndma-p-md
内側: .ndma-p-md
<div class="ndma-card ndma-p-md">...</div>
<div class="ndma-p-md">...</div>
<div class="ndma-m-md">...</div>
