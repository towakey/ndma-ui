# ndma-ui CSS デザイン指示書（Coding Agent 向け）

## 概要
**ndma-ui** は、架空の行政組織「国家災害管理機構（NDMA）」の業務UIを模した  
CSSフレームワークである。  
行政的で堅実なデザイン、災害対応向けのダークデザイン、さらに特務部隊向けの  
「夜間最適化テーマ（CENTRIS モード）」を含む複数テーマを提供する。

フレームワークは軽量で依存ライブラリなし。  
テーマ切替に強い、CSS変数中心の構成とする。

---

## 1. 提供すべきテーマ
### 1. Default（行政・文書系）
- 背景：白  
- NDMAブルー: `#102B4C`  
- アクセント：防災オレンジ `#E67A00`  
- 公的機関をイメージした控えめなUI  
- コンポーネントはアクセシビリティ優先

### 2. Dark（災害対策室向け）
- 黒/紺ベース  
- 透過感の少ないカード  
- 青と白でコントラストを確保

### 3. Night Ops（夜間最適化テーマ・CENTRIS UI）
- 完全ダークテーマ  
- NERVライクな発光オレンジ: `#FF8A1C`  
- 線的UI、HUDパネル、アニメーション有  
- CLT（危機管理連絡調整隊）の夜間作戦時に使用される「夜間最適化テーマ」

---

## 2. 実装すべきコンポーネント
### 基本
- Typography（h1〜h6, p）
- Grid / Column system（2〜12列）
- Card
- Form（input / select / textarea / checkbox）
- Button（primary / secondary / danger）
- Badge
- Alert
- Table
- Tabs
- Modal
- Banner（災害情報バー）

### NDMA 固有
- Status Tag（災害レベル）
- Mission Priority Badge
- 左側省庁ナビゲーション
- 災害対策室ヘッダ
- Monitor Panel（ダッシュボード要素）

### CENTRIS 専用（Night Opsテーマのみ）
- Tactical Button（輪郭＋発光）
- Grid HUD Panel
- Mini-map Frame
- Live feed window frame
- Warning flash animation（赤点滅）
- Code Entry Pad（認証UI）

## 3. カラートークン定義
```css
--ndma-blue: #102B4C;
--ndma-orange: #E67A00;
--ndma-gray: #6C7A89;
--ndma-light: #F5F7FA;

--ops-bg: #0A0A0A;
--ops-orange: #FF8A1C;
--ops-border: #E66000;
```

4. テーマ切替仕様
```
<html class="ndma-theme-default">
<html class="ndma-theme-dark">
<html class="ndma-theme-ops">
```

すべてのコンポーネントは
.ndma-theme-xxx をルートとしてスタイルを切替える。

5. コーディング規約

BEM準拠（例：.ndma-btn--primary）

外部JS不要、CSSのみ

CSS変数の利用必須

minified版も出力

WCAG AA 準拠（Night Opsは任意）

スマホ対応のレスポンシブ実装

6. デモページ index.html に必要な要素

NDMAヘッダ（ロゴ＋青バナー）

テーマ切替ボタン（Default / Dark / Night Ops）

Gridレイアウト例

災害状況リスト（Table）

Form（ログインフォーム）

Alert（緊急警報）

CENTRIS UI デモ（HUD風パネル）

Night Opsテーマでは、下記の演出をCSSだけで実装：

ホバー時のオレンジ発光

枠線の走査アニメーション

点滅Warningバー
