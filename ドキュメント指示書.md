# ② ndma-ui 公式ドキュメントサイト作成指示書（Markdown版）

```markdown
# ndma-ui 公式ドキュメントサイト 作成指示書
（Documentation Agent 向け）

## 概要
**ndma-ui** の公式ドキュメントサイトを制作する。  
目的は以下の通り：

- ユーザーが ndma-ui のコンポーネントを理解・利用できるようにする  
- デザイン哲学やテーマ構造を説明  
- サンプルコード・ライブデモを掲載  
- 世界観設定（NDMA / CENTRIS / Night Ops）を軽く添えることで  
  世界観 UI フレームワークとしての魅力も演出

サイトは静的HTML/CSS/JSで構成し、GitHub Pages でホスト可能とする。

---

## 1. サイト構造
以下の階層を基本とする。



docs/
├─ index.html
├─ getting-started.html
├─ components/
│ ├─ button.html
│ ├─ card.html
│ ├─ table.html
│ ├─ form.html
│ ├─ alert.html
│ └─ …
├─ themes/
│ ├─ default.html
│ ├─ dark.html
│ └─ ops.html
├─ examples/
│ ├─ dashboard.html
│ ├─ login.html
│ └─ centris.html
└─ assets/
├─ ndma.css
├─ ndma.min.css
└─ logo.svg


---

## 2. 必須ページの内容

### 2.1 トップページ `index.html`
- プロジェクト概要  
- デモGIF（任意）  
- インストール方法  
- テーマ切替サンプル  
- 特徴一覧（軽量、行政UI、CENTRISなど）

---

### 2.2 Getting Started
- CDN利用例  
- パッケージ導入例  
- 各テーマの切替方法  
- HTMLテンプレート例（最小構成）

---

### 2.3 Components
各ページには以下を必ず記載：

- 用途の説明  
- スクリーンショットまたはライブデモ  
- HTML例  
- カスタマイズ方法  
- テーマごとの差異表示

対象コンポーネント：
- Buttons  
- Cards  
- Forms  
- Tables  
- Alerts  
- Navigation  
- Badge / Tag  
- Modal  
- Grid  

---

### 2.4 Themes
#### default.html
- 行政系UI  
- 公的資料風の淡い色構成

#### dark.html
- 災害対策室UI  
- 濃紺ベースで視認性を強調

#### ops.html（Night Ops モード）
- HUD風  
- オレンジ発光  
- 線形UI  
- CENTRISで利用される特殊コンポーネントの紹介  
  - Tactical Button  
  - Live Panel  
  - Warning Flash  

---

### 2.5 サンプルページ（examples）
提供すべきデモページ：

#### dashboard.html（NDMA状況室）
- 災害アラート  
- 状況パネル  
- サマリーカード  

#### login.html
- フォーム・ボタンの実例

#### centris.html（特務システム画面）
- Night Opsテーマのみで構成  
- Tactical UI  
- HUD風グリッド  
- 認証パネル（CENTRISコード入力）

---

## 3. デザインガイド
- ドキュメントのUI自体も ndma-ui で構成する  
- 公式サイトのテーマ切替トグルを用意  
- クリーン・行政寄りのレイアウト  
- 目次（サイドナビ）を固定表示  
- コードブロックにはコピーボタンを配置  

---

## 4. 技術要件
- 静的HTMLのみ  
- JSはテーマ切替とUI補助のみ  
- モバイル対応  
- GitHub Pages でそのまま動作

---

## 5. チェックリスト
- [ ] 全ページでndma-uiが正しく適用されている  
- [ ] コンポーネントのコード例が正確  
- [ ] CENTRISページがNight Opsテーマの魅力を示す  
- [ ] グローバルナビで全ページを移動可能  
- [ ] 文書としての読みやすさ・構造が確保されている  
```
